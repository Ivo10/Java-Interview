### @Value

| 名称 | @Value                                               |
| ---- | ---------------------------------------------------- |
| 类型 | 属性注解  或  方法注解（了解）                       |
| 位置 | 属性定义上方  或  标准set方法上方  或  类set方法上方 |
| 作用 | 为  **基本数据类型**  或  **字符串类型**  属性设置值 |
| 属性 | value（默认）：要注入的属性值                        |
